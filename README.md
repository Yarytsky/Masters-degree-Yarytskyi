## Про проєкт

Веб-застосунок для шкільного навчального процесу, що поєднує функції електронного журналу з інструментами автоматизованого контролю знань та аналізу академічної доброчесності учнівських письмових робіт.

Реалізує метод інтегральної оцінки доброчесності за формулою 2.1 дисертації:

```
R = w₁·R_text + w₂·R_behav + w₃·R_stat
```

де `R_text` — ймовірність машинної генерації тексту, `R_behav` — поведінкові ознаки введення (вставка, паузи, правки), `R_stat` — стилометричне відхилення від попередніх робіт учня (профіль за предметом, формула 2.4).

Основні функції:

- журнал, розклад, оцінювання, домашні завдання;
- гібридний AI-детектор: локальний стилометричний аналіз (9 ознак — Yule's K, Honoré's R, MATTR, burstiness, ентропія пунктуації тощо) + семантичний аналіз через Claude API;
- антиплагіат між роботами класу (Жакар + триграмне перекриття);
- стилометричний профіль учня з фільтрацією AI-робіт із корпусу;
- захоплення поведінкової телеметрії в режимі реального часу (вставлений vs набраний текст);
- друкований PDF-звіт з повним розрахунком інтегральної оцінки.

Робоча демо-версія: **https://golden-jelly-6e60ed.netlify.app**

## Технології

- Frontend: vanilla JavaScript (18 модулів), CSS без фреймворків
- Backend: Netlify Functions (Node.js), проксі до Anthropic API
- AI: Claude Sonnet 4.6
- Хостинг: Netlify, auto-deploy з гілки `main`

## Запуск

### Варіант 1 — готова демо-версія (найшвидше)

Відкрити <https://golden-jelly-6e60ed.netlify.app> у браузері. Авторизація — вибір користувача зі списку. Для огляду повного функціоналу доброчесності увійти як вчитель.

### Варіант 2 — локальний запуск без AI

Потрібен Node.js.

```bash
git clone https://github.com/Yarytsky/Masters-degree.git
cd Masters-degree
npm start
```

Відкрити <http://localhost:3000>. Працюють усі функції крім семантичного аналізу через Claude API (для нього потрібен Варіант 3).
